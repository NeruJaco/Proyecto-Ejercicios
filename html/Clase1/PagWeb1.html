Definición de:

HTML

[HyperText Markup Language] Es la quinta y más reciente versión del Lenguaje de Marcado de Hipertexto (HTML). Es el estándar fundamental para crear y estructurar el contenido de las páginas web, controlando cómo los navegadores muestran textos, imágenes y elementos interactivos.

CSS3

[Cascading Style Sheets level 3] (Hojas de Estilo en Cascada) es un lenguaje de diseño web utilizado para describir la apariencia y el formato visual de documentos escritos en HTML o XML. Sirve para controlar la presentación (colores, fuentes, márgenes) separándola del contenido, lo que facilita la creación de sitios web modernos y adaptables.

JavaScript

Es un lenguaje de programación interpretado y ligero, diseñado principalmente para dotar de interactividad y dinamismo a las páginas web. Es la tecnología que permite crear animaciones, mapas interactivos, ventanas emergentes y actualizar datos sin necesidad de recargar el navegador.
